
Caderno Temático: Big Data
Contexto e Objetivos

Este projeto tem como objetivo estudar o conceito de Big Data, suas aplicações e importância no mundo atual.

Os principais objetivos são:

Entender o que é Big Data
Identificar suas principais características
Conhecer aplicações no dia a dia
Compreender sua importância para empresas e tecnologia
Curadoria de Fontes

As fontes utilizadas para estudo foram:

Engenharia de Prompts e Testes
Durante o estudo, foram utilizados os seguintes prompts:

Prompt 1:

O que é Big Data?

Resposta:
Big Data refere-se a grandes volumes de dados que não podem ser processados por métodos tradicionais.

Prompt 2:

Para que serve o Big Data?

Resposta:
Serve para analisar dados e gerar informações importantes para tomada de decisões.

Prompt 3:

Dê exemplos de aplicações de Big Data

Resposta:

Netflix recomendando filmes
Bancos detectando fraudes
Redes sociais analisando comportamento

Dificuldades (Cicatrizes)
Algumas respostas eram muito genéricas
Foi necessário refazer perguntas para obter respostas mais completas
Aprendi que prompts mais específicos geram respostas melhores

Miniguia de Estudo
Resumo

Big Data é o conjunto de dados extremamente grandes que precisam de tecnologias específicas para serem analisados.

Se baseia nos 5 V's:

Volume
Velocidade
Variedade
Veracidade
Valor
Glossário
Big Data: grande volume de dados
Dados estruturados: organizados em tabelas
Dados não estruturados: como vídeos e textos
Análise de dados: processo de interpretar dados
